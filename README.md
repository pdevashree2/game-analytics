# Game Analytics Dashboard
A data analytics project built with Python, SQL, and Streamlit.

## Live Demo
https://pdevashree2-game-analytics-app.streamlit.app

## Tech Stack
- Python, SQLite, Pandas, Plotly, Streamlit

## Key Metrics
- Total Players, Sessions and Revenue
- Players by Country and Platform
- Most Played Games
- Revenue by Game and Country
- Top 10 Players by Score
- Daily Active Users

## Run Locally
git clone https://github.com/pdevashree2/game-analytics.git
cd game-analytics
pip install -r requirements.txt
python generate_data.py
streamlit run app.py
